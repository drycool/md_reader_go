# Проект MD Reader (Go)
Это интерактивный просмотрщик документации Markdown на языке Go.

## Функции
- Быстрый нечеткий поиск по заголовкам.
- Удобное дерево навигации.
- Рендеринг текста прямо в окне.

## Как пользоваться
1. Введите запрос в поле поиска слева.
2. Выберите нужный раздел из списка.
3. Читайте содержимое в правой части окна.

### Системные требования
- Go 1.22+
- C-компилятор (для графики)
